## 使用方法
1. index.htmlをブラウザで開く
2. テキストエリアにHTMLを入力する
   1. `samples/`にサンプル用のHTMLファイルを配置しています。
3. 「ツリーを更新」ボタンをクリックする
4. ツリー表示とHTML表示が更新される
